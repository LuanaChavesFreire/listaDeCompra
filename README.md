# 🛒 Lista de Compras

Uma aplicação simples de **Lista de Compras** desenvolvida com **HTML, CSS e JavaScript puro**, criada para praticar manipulação de DOM, organização de código e lógica de programação.

## 📌 Sobre o Projeto

Este projeto foi desenvolvido com o objetivo de fortalecer conceitos fundamentais de JavaScript por meio da construção de uma aplicação interativa do zero.

A aplicação permite adicionar itens a uma lista de compras de forma dinâmica, atualizando a interface em tempo real.

O código foi organizado de maneira modular para manter a estrutura limpa e facilitar a manutenção.

## 📂 Estrutura do Projeto

Principais arquivos do projeto:

- `index.html` — Estrutura principal da interface  
- `styles.css` — Estilização da aplicação  
- `script.js` — Lógica principal para adicionar itens  
- `/scripts/` — Scripts auxiliares e organização modular  

## 🛠️ Funcionalidades

- ➕ Adicionar novos itens à lista
- 📦 Atualização dinâmica da interface
- 🧩 Organização modular do JavaScript
- 🔎 Verificação de lista vazia com feedback visual
